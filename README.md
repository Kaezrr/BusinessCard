# Business-Card-App
 A fully customizable Business Card App based on the Android Basics Course from Google written in Kotlin with Android Studio.
 
 Includes: <br>
 -Name <br>
 -Title <br>
 -Phone Number <br>
 -Social Handles <br>
 -Email <br>
